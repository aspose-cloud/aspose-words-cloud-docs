This REST API updates (reevaluates) fields in a document.

## REST API

The following URIs are used to address REST resources:

```HTML
~/{file-name}/updateFields
~/{file-name}/updateFields?filename={DestDocumentName}
```

The [OpenAPI Specification](https://apireference.aspose.cloud/words/#/Fields/UpdateFields) lets you call this REST API directly from a browser.

You can also use cURL command-line utility to test this REST API. The following are a few examples of using cURL.

{{< nosnippet >}}
{{< tabs tabTotal="2" tabID="2" tabName1="Request" tabName2="Response" >}}
{{< tab tabNum="1" >}}

```bash
# cURL example to update Fields in a Document
curl -v "https://api.aspose.cloud/v4.0/words/test_multi_pages.docx/updateFields" \
-X PUT \
-H "Content-Type: application/json" \
-H "Accept: application/json" \
-H "Content-Length: 0" \
-H "Authorization: Bearer <jwt token>"
```

<p style="margin:0;font-size:80%;font-style:italic">To get a jwt token use this <a href="/words/getting-started/available-sdks/#curl">instruction</a></p>
